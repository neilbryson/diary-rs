# diary-rs

## Requirements
- [Rust](https://www.rust-lang.org/)
- `libsqlite3-dev` (install from package manager)

## Run locally
```bash
cargo run -- --ARGS

# Example, to run with the --help flag
cargo run -- --help
```

## Building from source
To download Rust crates and compile the program, run :

```bash
cargo build
```
